```
│   ├── users.ts           # 用户 API
│   ├── messages.ts        # 私信 API
│   ├── follow.ts          # 关注 API
│   ├── notifications.ts   # 通知 API
│   ├── interactions.ts    # 互动 API
│   ├── upload.ts          # 上传 API
│   └── resources.ts       # 学习资料 API
├── store/                  # 状态管理
│   └── authStore.ts       # 用户认证状态
├── types/                  # TypeScript 类型
│   └── api.ts             # API 响应类型
├── components/             # 组件
│   ├── Layout.tsx         # 主布局
│   └── ProtectedRoute.tsx # 受保护路由
├── pages/                  # 页面
│   ├── Home.tsx           # 首页
│   ├── Login.tsx          # 登录
│   ├── Register.tsx        # 注册
│   ├── PostDetail.tsx     # 帖子详情
│   ├── CreatePost.tsx     # 创建帖子
│   ├── EditPost.tsx       # 编辑帖子
│   ├── UserProfile.tsx    # 用户中心
│   ├── EditProfile.tsx    # 编辑资料
│   ├── Search.tsx         # 搜索
│   ├── Messages.tsx       # 消息列表
│   ├── Conversation.tsx   # 对话页面
│   ├── Notifications.tsx  # 通知列表
│   ├── Resources.tsx      # 学习资料列表
│   └── UploadResource.tsx # 上传资料
├── App.tsx                 # 应用入口
└── main.tsx               # 入口文件
```

## 🛠️ 技术特性

### 状态管理
- **Zustand** - 轻量级状态管理
- **持久化存储** - localStorage 自动同步
- **React Query** - 服务端状态管理

### 路由系统
- **React Router v7** - 路由管理
- **受保护路由** - 需要登录的页面
- **路由守卫** - 自动跳转登录

### UI/UX
- **Tailwind CSS 4** - 现代化样式
- **响应式设计** - 移动端适配
- **Lucide React** - 图标库
- **加载状态** - 友好的加载提示
- **错误处理** - 统一的错误提示

### 表单处理
- **React Hook Form** - 表单管理
- **Zod** - 表单验证
- **实时验证** - 即时反馈

## 🚀 运行方式

### 开发环境

1. **启动后端服务**
   ```bash
   cd backend
   npm run dev
   ```

2. **启动前端服务**
   ```bash
   cd frontend
   npm run dev
   ```

3. **访问应用**
   - 前端：http://localhost:5173
   - 后端 API：http://localhost:3000/api

### 环境变量

创建 `frontend/.env.local` 文件：
```env
VITE_API_URL=http://localhost:3000
```

## 📝 注意事项

### 开发环境
- 前端运行在 `http://localhost:5173`
- 后端 API 运行在 `http://localhost:3000`
- 确保后端服务已启动

### Token 管理
- Token 存储在 localStorage
- 自动添加到请求头
- Token 过期自动清除并跳转登录

### 实时更新
- 私信页面每3秒自动刷新
- 通知/消息未读数每30秒刷新
- 使用 React Query 缓存和更新

### 类型安全
- 所有 API 响应都有 TypeScript 类型定义
- 表单验证使用 Zod schema
- 组件 Props 都有类型定义
