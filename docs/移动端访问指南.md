## 📱 如何在手机上访问

### 方法一：本地开发测试（同一网络）

#### 1. 确保手机和电脑在同一WiFi网络

#### 2. 启动开发服务器

```bash
cd frontend
npm run dev
```

#### 3. 查看电脑的IP地址

**Windows:**
```powershell
ipconfig
# 查找 IPv4 地址，例如：192.168.1.100
```

**macOS/Linux:**
```bash
ifconfig
# 或
ip addr show
```

#### 4. 在手机上访问

- **打开手机浏览器**（Chrome或Safari）
- **访问：** `http://your-computer-ip:5173`
  - 例如：`http://192.168.1.100:5173`

#### 5. 添加到主屏幕

**iOS Safari:**
1. 点击底部的"分享"按钮
2. 选择"添加到主屏幕"
3. 输入名称（默认"校园论坛"）
4. 点击"添加"

**Android Chrome:**
1. 点击右上角菜单（三个点）
2. 选择"添加到主屏幕"或"安装应用"
3. 确认添加

---

### 方法二：部署后访问（推荐）

部署到服务器后，使用域名访问：

1. **访问部署的网站**
   - 例如：`https://yourdomain.com`

2. **添加到主屏幕**
   - iOS：分享 > 添加到主屏幕
   - Android：菜单 > 添加到主屏幕

3. **像原生APP一样使用**
   - 从主屏幕打开
   - 全屏显示（无浏览器地址栏）
   - 支持离线访问（如果配置了）

## 💡 常见问题

### Q: 手机上无法访问localhost？
A: 使用电脑的IP地址，不是localhost。例如：`http://192.168.1.100:5173`

### Q: 手机和电脑不在同一网络？
A: 可以使用以下方法：
- 使用手机热点（电脑连接手机热点）
- 使用ngrok等工具（创建公网隧道）
- 部署到服务器后访问

### Q: iOS Safari无法添加到主屏幕？
A: 确保：
- 使用Safari浏览器（不是Chrome）
- 网站使用HTTPS（或localhost）
- Manifest配置正确

### Q: 添加到主屏幕后图标不显示？
A: 确保：
- `icon-192.png` 和 `icon-512.png` 文件存在
- 文件路径正确（`/icon-192.png`）
- 文件格式正确（PNG）
