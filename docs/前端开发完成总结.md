## 🛠️ 已实现功能

### 1. API 客户端配置 ✅

- **Axios 实例配置**
  - 基础 URL 配置（支持环境变量）
  - 请求拦截器（自动添加 Token）
  - 响应拦截器（统一错误处理）
  - Token 过期自动跳转登录

- **API 模块**
  - `authApi` - 认证相关（注册、登录、获取用户、登出）
  - `postsApi` - 帖子相关（列表、详情、创建、更新、删除、搜索）
  - `categoriesApi` - 分类相关（列表、详情）
  - `commentsApi` - 评论相关（列表、创建、更新、删除）

### 2. 状态管理 ✅

- **Zustand Store**
  - `useAuthStore` - 用户认证状态管理
  - 持久化存储（localStorage）
  - 自动同步 Token
  - 登录/注册/登出功能
  - 获取用户信息

### 3. 路由系统 ✅

- **React Router 配置**
  - 公开路由：`/login`, `/register`
  - 需要布局的路由：`/`, `/posts/:id`
  - 404 重定向
  - 受保护路由组件（`ProtectedRoute`）

### 4. 基础组件 ✅

- **Layout 组件**
  - Header（导航栏）
    - Logo 和标题
    - 导航菜单（首页、搜索）
    - 用户菜单（头像、用户名、登出）
    - 发帖按钮（登录后显示）
    - 登录/注册按钮（未登录显示）
  - Footer（页脚）
  - 响应式设计

- **ProtectedRoute 组件**
  - 登录状态检查
  - 未登录自动跳转
  - 加载状态显示

### 5. 页面开发 ✅

#### 登录页面 (`/login`)
- 表单验证（React Hook Form + Zod）
- 邮箱和密码输入
- 错误提示
- 跳转到注册页链接
- 登录成功后跳转首页

#### 注册页面 (`/register`)
- 表单验证（邮箱、用户名、密码、确认密码）
- 错误提示
- 跳转到登录页链接
- 注册成功后跳转首页

#### 首页 (`/`)
- 帖子列表展示
- 分类侧边栏
- 帖子卡片（标题、内容预览、作者、时间、统计）
- 分类筛选
- 加载状态
- 空状态提示

#### 帖子详情页 (`/posts/:id`)
- 帖子完整内容展示
- 图片展示（如果有）
- 作者信息
- 统计数据（浏览量、评论数、点赞数）
- 评论列表
- 发表评论功能
- 回复评论功能
- 编辑/删除按钮（仅作者可见）
- 返回按钮

## 🚀 运行方式

### 开发环境

1. **启动后端服务**
   ```bash
   cd backend
   npm run dev
   ```

2. **启动前端服务**
   ```bash
   cd frontend
   npm run dev
   ```

3. **访问应用**
   - 前端：http://localhost:5173
   - 后端 API：http://localhost:3000/api

### 环境变量

创建 `frontend/.env.local` 文件：
```env
VITE_API_URL=http://localhost:3000
```
