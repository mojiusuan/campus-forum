# PWA图标说明

## 📱 需要的图标文件

请创建以下图标文件并放在 `public/` 目录：

- `icon-192.png` (192x192像素)
- `icon-512.png` (512x512像素)

## 🎨 生成图标的方法

### 方法1：使用在线工具（最简单）

1. 访问：https://realfavicongenerator.net/
2. 上传你的Logo图片
3. 选择"Generate favicons and HTML code"
4. 下载生成的图标包
5. 提取 `android-chrome-192x192.png` 和 `android-chrome-512x512.png`
6. 重命名为 `icon-192.png` 和 `icon-512.png`
7. 放到 `public/` 目录

### 方法2：使用设计工具

使用 Photoshop、Figma、Canva 等：
1. 创建512x512的画布
2. 设计应用图标（建议使用圆角矩形背景）
3. 导出为PNG
4. 调整尺寸到192x192和512x512

### 方法3：使用提供的生成器

打开 `generate-icons.html` 在浏览器中，点击生成和下载按钮。

## ⚠️ 临时占位符

在创建真实图标之前，你可以：
1. 使用任何192x192和512x512的PNG图片作为占位符
2. 或者暂时使用纯色图片

## 📝 图标设计建议

- **尺寸：** 192x192 和 512x512
- **格式：** PNG（支持透明）
- **设计：** 简洁、易识别
- **背景：** 建议使用品牌色（#3b82f6）
- **文字：** 可以使用"论"字或Logo
